# Данные рейтинга

Основной источник данных — `ratings-2026.xlsx`. После сохранения файла
достаточно обновить страницу сайта — данные подтянутся автоматически.

Структура Excel:
- можно использовать 1 лист (данные будут применены ко всем месяцам);
- или 12 листов с названиями месяцев: Январь–Декабрь;
- колонка `Вуз` (полное название);
- числовые колонки: `Сводный рейтинг`, `СМИ`, `Социальные сети`,
  `ВКонтакте`, `Telegram`, `MAX`, `Rutube`, `Сайт`, `Федеральная повестка`;
- если `Социальные сети` пустая, итог считается суммой `ВКонтакте`+`Telegram`+`MAX`+`Rutube`.

Автоматическая генерация JSON из Excel:
- скрипт `scripts/build_ratings_json.py` создаёт `ratings-2026.json`;
- JSON можно использовать для предпросмотра или публикации.
